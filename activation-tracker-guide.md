# Denuvo Activation Tracker Guide

Use this tracker to stay within a 5-activations-per-day limit.

## How to use

1. Open the CSV file in Excel, Google Sheets, or LibreOffice.
2. Add one row each time you activate or re-activate the game.
3. Keep `Daily Count Used` accurate for the same date.
4. Stop activating when count reaches 5 for that date.
5. If an activation fails, still log it in case support asks for details.

## Suggested columns

- Date: Activation date (YYYY-MM-DD)
- Time: Activation time (24h)
- Game: Game title
- Store/Launcher: Steam, Epic, EA App, etc.
- Device Name: PC name or device label
- OS/Hardware Change?: Yes/No
- Activation Type: New or Re-activation
- Result: Success or Fail
- Daily Count Used: Running count for that date
- Notes: Anything useful for support

## Support checklist

If you hit a lockout or lose an activation, collect:

- Purchase receipt
- Account email
- Date and time from tracker
- Device details
- Error message screenshot
